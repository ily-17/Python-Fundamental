# Python-Fundamental
this git repo contains of all the Python fundamentals

## Overview

This repository contains comprehensive Python fundamentals examples, exercises, and learning materials designed for beginners and aspiring software developers. The goal is to build a strong foundation in Python programming, covering core concepts, object-oriented programming, file handling, APIs, testing, and development best practices.

## Topics Covered

### Python Fundamentals I

#### Python Setup
*Installing Python

Download and install Python from the official Python website.
Verify the installation using:

   python --version
   
Understand Python versions and environment configuration.

*Setting Up VS Code / PyCharm

Install a Python-compatible IDE or code editor.
Configure the Python interpreter.
Install useful extensions/plugins for Python development.
Learn basic IDE features such as debugging, terminal usage, and code navigation.

*Running Python Scripts

Execute Python files from the command line:

python hello.py

Run programs directly from VS Code or PyCharm.
Understand script execution flow and entry points.

*Python REPL and IDE Usage

Use the Python REPL (Read-Eval-Print Loop) for quick experimentation:

python
print("Hello World")
Practice executing statements interactively.
Learn the differences between interactive mode and script execution.
Explore IDE features such as code completion, syntax highlighting, debugging, and integrated terminals.
