# Calculator Package

## Overview

This project demonstrates how to organize a Python application into reusable modules and packages.

## Project Structure

```
calculator_package/
│
├── README.md
├── requirements.txt
├── main.py
│
└── calculator/
    ├── __init__.py
    ├── operations.py
    └── utils.py
```

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Menu-driven interface
- Modular code structure

## Run the Project

```bash
python main.py
```

## Concepts Covered

- Python Modules
- Python Packages
- Imports
- Project Structure
- Code Reusability
